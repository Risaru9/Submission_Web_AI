# Root Fact App

Aplikasi React untuk mengenali sayuran dari kamera menggunakan TensorFlow.js dan membuat fun fact dengan Transformers.js.

## Menjalankan Proyek

```bash
npm install
npm run dev
```

## Build Production

```bash
npm run lint
npm run build
```

Deploy folder `dist` ke Netlify, lalu isi `Deployment URL` pada `STUDENT.txt`.
